# Informal Proofs

This directory contains structured informal proofs generated by the `/prove-missing`
workflow for theorems that are cited without proof in the Bubeck book.

Each file is named `<TheoremName>_informal.md` and contains:
- Theorem statement
- Sources found via WebSearch
- Step-by-step proof (English + Math + Lean/Mathlib hints)
- Mathlib gaps identified during formalization
- Status: PROOF FOUND | CLAUDE-GENERATED | PROOF NOT FOUND
